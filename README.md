# Metagascar

**The Unfungable Metaverse**

Metagascar is an Ethereum NFT collection and metaverse concept built around scarce digital identity, collectible artwork, and the playful idea that some places, characters, and stories are too distinct to be treated as interchangeable.

The collection is listed on OpenSea as **Metagascar Westside**.

## Links

- Website: https://metagascar.com

## What "Unfungable" Means

Metagascar uses "unfungable" as a brand idea: a world where the value is not only in ownership, but in distinctiveness. Each collectible is part of a broader fictional metaverse identity rather than just another interchangeable asset.
